# Scrapper do Mercado Livre
Autor: Iury Goulart

## Projeto para adquirir dados do Mercado Livre para pegar produtos piratas

Já está com o ambiente python virtual: `./.mercado-livre-env/bin/activate`

Foi utilizado `python 3.10.14` 

### Bibliotecas usadas:
- beautifulsoup4
- numpy
- Pillow
- selenium
- webdriver_manager
- json
